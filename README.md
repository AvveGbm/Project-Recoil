# Project Recoil
#### Summary
Project Recoil is a 2D-platformer where traversal happens entirely through recoil! Use careful timing and correct your trajectory to soar your way to the finish line: in style! Strategically manage your ammo to ensure you make it to the finish line whilst increasing your score with collectable coins. Just don't spend too long planning your next move, lest the time on the clock begins to look less than impressive!

The project is created in version 4.6 of the Godot game engine, using GDScript as the primary language.

#### Minimum Viable Product
The minimum viable product is slightly above the minimum requirements of a game. The MVP has interactability, a goal, and some additional minor features. Concretely, this means a means of moving the player, a working environment in which the player can move (levels) and level essentials, e.g. death areas, a finish line, and pick-ups.

#### Installation and Setup 
##### Run in engine editor
- Download the standard version of **Godot 4.6** (or later) from https://godotengine.org/ 
- Clone the repository
- Import the cloned repo in Godot project manager
- Open the project and press play to start the main scene
  
##### Run latest release (executable, Windows only)
- Go to [releases](https://github.com/AvveGbm/Project-Recoil/releases) and download the latest available release (only the .exe)
- Run Project-Recoil.exe
- Pray the malware on our computers doesn't propagate to yours ;) <sub>(for legal reasons, this is a joke)</sub>

#### How to play
Aim your weapon using your mouse, and left-click to fire and push yourself with the recoil of your shot. Get to the end of the level without running out of ammunition. ESC brings up the pause menu.
